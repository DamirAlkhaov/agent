Version 1.4.0 (PRE-ALPHA)
--------------------------------------------------------------------------------------------------------------------------------------------------------
*) Cleaned up flight tracking UI
   *) Moved detailed fuel and payload management into new expandable column (hidden on default)
   *) Landing report is now a standalone notification - you can choose when to show it in the settings
*) Full debug logging implemented (in case I need to figure out what happened in case you run into a bug)
*) Set time + plane registration in sim implemented
*) Full flight saving+restoring implemented ... click on stop tracking and select Yes you want to save in the dialog
   *) Save files are compressed, but for now at least a XML version is saved alongside (those also don't get deleted if you cancel)
   *) Also auto-saves every 2 minutes in case of tracker crash
   *) Saves if connection to sim is lost
*) Slew plane into position implemented
   *) Can move to origin airport location (if spawned in wrong location or sim doesn't have the airport)
   *) For resuming flights - comes with instruction box, can you check if that is explained well enough
*) Resume flight, two ways at the moment
   *) Resume straight from flight tracking view, after stopping tracking
   *) If you closed the tracker, so this includes resuming on the next day etc. - you need to create the same flight using the
      test flight window (best to screenshot your old settings) + TICK THE "Resume flight?" checkbox - This would normally all come
      from the backend, but for now this how to do it
*) Added text-to-speech for lots of tracking start/stop events
   *) Voice can be changed in the settings
*) Some feedback sound effects
*) Known issue: The ground handling background worker needs overhauled, if you close the tracking window and re-open it, it will
   spawn another thread (to update the UI on this window, but it will play the audio twice, I fill overhaul that in the next version)
*) A million and one small and large fixes :) - so try to brake it please
